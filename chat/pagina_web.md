# Guía para crear tu primera página web con ChatGPT

## Introducción

Este documento está pensado para quienes quieren crear su **primera página web**, sin tener experiencia técnica previa. El objetivo es que puedas hacerlo paso a paso junto con ChatGPT, entendiendo cómo se organizan los archivos, en qué carpeta deben ir y cómo ver el resultado en tu navegador.  
Vas a aprender cómo se estructura un sitio web moderno con **HTML, CSS y JavaScript**, usando **Bootstrap** para que se vea bien en computadoras y celulares. El diseño será **sobrio, elegante y minimalista**, ideal para un catálogo de ropa femenina.

El resultado final será una web completa con tres secciones —**Inicio**, **Catálogo** y **Contacto**—, un menú superior para navegar, un pie de página con el año actual, y un formulario de contacto que incluye un botón de WhatsApp para que tus clientes puedan comunicarse.

---

## Prompt para ChatGPT

> Hoy, una tarea sencilla para vos qué sos un desarrollador web súper destacado y profesional. Voy a querer que diseñes y escribas el código de una página web en HTML, CSS y Javascript. Nada más. Debe tener una página de inicio y un menú horizontal superior con tres pestañas: Inicio, Catálogo, Contacto. Cada pestaña debe llevar a una página con el contenido adecuado. Quiero que sea sobria, elegante y minimalista. Debe tener un pie de página con el año. Es para mostrar mí catálogo de ropa de mujer. Quiero que inventes el contenido por ahora. En catálogo quiero 6 cajas con Foto descripción y precio. Y en contacto un formulario y un botón que abra WhatsApp. Después te doy el número para que funcione. Usa bootstrap para la maqueta y que sea responsive. Dame los descargables y decime en qué carpeta pongo cada cosa.

---

## Estructura de carpetas

Creá la carpeta principal en tu computadora dentro de **Documentos** con el siguiente nombre:

`Documentos/mi_pagina_web/`

Dentro, organizá los archivos así:

```
mi_pagina_web/
├── index.html              ← Página de inicio
├── catalogo.html           ← Catálogo de productos
├── contacto.html           ← Formulario de contacto
├── css/
│   └── estilos.css         ← Estilos personalizados
├── js/
│   └── script.js           ← Funcionalidades básicas
└── img/
    ├── producto1.jpg
    ├── producto2.jpg
    ├── producto3.jpg
    ├── producto4.jpg
    ├── producto5.jpg
    └── producto6.jpg
```

---

## Recomendaciones

1. Guardá los archivos con los nombres exactos indicados.  
2. Para ver tu sitio, abrí el archivo **index.html** con tu navegador (doble clic o clic derecho → “Abrir con”).  
3. Podés cambiar las fotos dentro de la carpeta **img/** cuando tengas tus propias imágenes.  
4. Modificá colores o fuentes dentro de **css/estilos.css**.  
5. En **contacto.html**, reemplazá el número ficticio por el tuyo para activar el botón de WhatsApp.
